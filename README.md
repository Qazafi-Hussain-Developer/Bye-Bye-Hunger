# 🍽️ Bye-Bye-Hunger

A full-stack **Restaurant Management System** built using **React, Node.js, Express, MongoDB, and MySQL**.

## 🚀 Features

### User Features

- User Registration & Login (JWT)
- Email Verification
- Password Reset via Email
- Browse Food Menu
- Add to Cart
- Place Orders
- Order Tracking
- Loyalty Points
- Order Rating System

### Admin Features

- Admin Dashboard
- Manage Foods (CRUD)
- Manage Orders
- Manage Users
- View Customer Reviews

### Technical Features

- JWT Authentication
- Role-based Access Control
- Email Notifications
- File Upload with Multer
- Dual Database (MongoDB + MySQL)

## 🛠 Tech Stack

Frontend

- React.js
- Vite
- Axios
- CSS

Backend

- Node.js
- Express.js
- MongoDB
- MySQL
- JWT Authentication
